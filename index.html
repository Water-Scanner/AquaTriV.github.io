AquaTriV

A Unified Multi-Modal Underwater Dataset for Dense SLAM and Neural Reconstruction

25 sequences · 255GB · 6.9h · 2.8km trajectory

Yaming Ou · Junfeng Fan · Xiaoyan Liu · Chao Zhou · Pengjv Zhang

CASIA · (More collaborators)

Paper Code Dataset

Multi-scene underwater data acquisition across pool, cave, river, and offshore environments.

Overview

Multi-Modal Sensing

Passive, active, neuromorphic vision + IMU, DVL, pressure

Dense Ground Truth

Laser-scanned dense point clouds for mapping evaluation

Accurate Localization

MCS (indoor) + INS-SfM fusion (outdoor)

Key Contributions

  • First underwater dataset combining active, passive, neuromorphic vision
  • High-fidelity dense point clouds for SLAM & neural reconstruction
  • New evaluation metrics: ATE60, CRS, TCR
  • Multi-scene coverage: pool, cave, river, sea
